# IntelliDoc 

**AI-Powered Question & Answer Generator**

IntelliDoc is an interactive AI-driven application that automatically generates meaningful questions and answers from documents using **Natural Language Processing (NLP)** and **pre-trained Large Language Models (LLMs)**. It is designed to assist educators and students by saving time in assessment and content creation.

---

##  Features

*  Accepts textual documents as input
*  Extracts text using **Tesseract OCR**
*  Generates context-aware questions and answers using **QuestGenAI**
*  Fast, automated, and easy to use
*  Ideal for educational and academic use cases

---

##  Tech Stack

* **Language:** Python
* **OCR:** Tesseract-OCR
* **NLP / LLM:** QuestGenAI (Pre-trained model)
* **Other Tools:** NLP processing libraries

---

##  How It Works

1. Upload a document (text or scanned image).
2. Extract text from the document using **Tesseract OCR**.
3. Clean and preprocess the extracted text.
4. Generate relevant questions and answers using **QuestGenAI**.
5. Display the generated output in an interactive format.

---

##  Use Cases

* Automated question paper generation
* Study material creation
* Faculty assessment support
* Student self-evaluation
